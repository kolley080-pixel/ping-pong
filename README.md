En el repositorio se encuentra el prototipo del juego «Ping-pong». Se trata de un juego multijugador.
El juego está programado en Python utilizando la biblioteca Pygame.
El jugador 1 controla la raqueta izquierda con las teclas
El jugador 2 controla la raqueta derecha con las teclas
Que tengas un buen partido :D
